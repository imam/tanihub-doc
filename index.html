    Introduction

    Welcome to the TaniHub API! You can use our API to access TaniHub API endpoints, which can get information on various book and book categories in our database.

    We have language bindings in Shell! You can view code examples in the dark area to the right, and you can switch the programming language of the examples with the tabs in the top right.

    This example API documentation page was created with Slate. Feel free to edit it and use it as a base for your own API’s documentation.

    Insert Book

    This endpoint is to input Book.

    curl "http://tanihub.dev/api/book"\
-H "Content-Type: application/json" \
-d '{"book_category_id": "2", "title": "Halo", "author": "Imam Assidiqqi"}'

    The above command returns JSON structured like this:

    {
  "book": {
    "id": 54,
    "title": "Halo",
    "author": "Imam Assidiqqi",
    "book_category": {
      "id": 2,
      "description": "Vel aliquid rerum dicta quia amet velit. Quae qui neque ut iure. Sed culpa dolor aut quis non id.",
      "created_at": "2017-02-16 12:34:18",
      "updated_at": "2017-02-16 12:34:18"
    }
  }
}

    HTTP Request

    POST http://tanihub.dev/api/book

    Payload

    Field Data Type Mandatory Description
    book_category_id Integer Yes Book category Id
    title String Yes Book Title
    author String Yes Book Author

    Update Book

    This endpoint is to update Book.

  -H "Content-Type: application/json" \

    "book_category": {

    Errors

    This error section is stored in a separate file in `includes/_errors.md`. Slate allows you to optionally separate out your docs into many files…just save them to the `includes` folder and add them to the top of your `index.md`’s frontmatter. Files are included in the order listed.

    The Kittn API uses the following error codes:

    Error Code Meaning
    400 Bad Request – Your request sucks
    401 Unauthorized – Your API key is wrong
    403 Forbidden – The kitten requested is hidden for administrators only
    404 Not Found – The specified kitten could not be found
    405 Method Not Allowed – You tried to access a kitten with an invalid method
    406 Not Acceptable – You requested a format that isn’t json
    410 Gone – The kitten requested has been removed from our servers
    418 I’m a teapot
    429 Too Many Requests – You’re requesting too many kittens! Slow down!
    500 Internal Server Error – We had a problem with our server. Try again later.
    503 Service Unavailable – We’re temporarially offline for maintanance. Please try again later.
